# Documentation

This folder is the primary documentation for the `anatomy` project.

Key documents:

- `ARCHITECTURE.md` — high level architecture and component references.
- `QUICKSTART.md` — setup and example run commands.
- `API_REFERENCE.md` — where to find HTTP API route implementations.
- `LAYERS/` — per-layer design notes and code references.

Use the `EXAMPLES.md` to find runnable examples and their command lines.
